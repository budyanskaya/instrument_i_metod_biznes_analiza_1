# Инструменты и методы бизнес-анализа
## Задание 1
Используя нотацию BPMN и Bizagi Modeler, сформируйте и продемонстрируйте модель бизнес-процессов  TO-BE для автоматизации работы с клиентами в онлайн школе. Выполните анализ улучшений модели TO-BE (по сравнению с моделью AS-IS) по экономическим параметрам и по степени ее готовности для создания сценария интеграции.

## Решение
<img width="1201" height="691" alt="Инструменты и методы бизнес-анализа 1 drawio" src="https://github.com/user-attachments/assets/d36a2b8a-0a8b-43eb-9da2-9c0b1f55d02f" />
На BPMN-диаграмме «Онлайн-школа (TO-BE)» описан автоматизированный процесс работы с клиентами: клиент оставляет заявку, система регистрирует лида и проводит скоринг, затем либо автоматически генерирует ссылку на оплату, либо передаёт клиента менеджеру для консультации. После оплаты система создаёт аккаунт в LMS, куратор звонит клиенту, и клиент начинает обучение.

По сравнению с AS-IS модель TO-BE даёт экономию: сокращение времени обработки заявок на 80-90%, снижение операционных расходов на 60-70%, повышение конверсии в оплату на 15-25% за счёт автоматизации скоринга, генерации ссылок, создания аккаунта и чёткого цикла обработки возражений.

Готовность модели для интеграции — 4 из 5. Есть чёткие зоны ответственности, автоматические шаги и временной таймаут, но не хватает обработки ошибок интеграции (сбой LMS, недоступность платежей), протоколов обмена и длительности задач. Модель пригодна для ТЗ, но требует доработки.
